# Moder_for_BPM_PRIME

Telegram-бот для модерации и фарма для группы BPM PRIME.

## 🚀 Возможности

### 🛡 Модерация
- **Чёрный список слов** — ловит слова в любом регистре, с разрывами (`п р и в е т`, `П.Р.И.В.Е.Т`, `ПрИвЕт`)
- **Бан/Мут/Варн** — полный набор команд для админов
- **Автоматические действия** — предупреждение, мут, бан, удаление сообщения
- **Логи модерации** — история всех действий

### 💰 Фарминг 
- **Ежедневка** — бонус каждый день + бонус за стрик (до 30 дней)
- **Работы** — 11 работ от Разнорабочего до Техно-магната с прогрессией по уровням
- **Преступления** — 8 преступлений с риском тюрьмы (от кражи велосипеда до кибератаки)
- **Бизнесы** — 7 типов бизнесов для пассивного дохода (кофейня → дата-центр)
- **Кланы** — создание кланов, казна, топ кланов, офицеры
- **Магазин** — предметы: бусты, защита от мута/бана, снятие варнов, клановые предметы
- **Инвентарь** — использование купленных предметов

### ⚙️ Админка
- Статистика бота
- Рассылка по всем пользователям
- Выдача денег/опыта/уровней
- Принудительное обновление кэша чёрного списка

## 🛠 Технологии
- **Python 3.11+**
- **aiogram 3.x** — современный асинхронный фреймворк для Telegram Bot API
- **SQLAlchemy 2.0** (async) + **asyncpg** — работа с PostgreSQL
- **PostgreSQL** (Supabase/Neon) — база данных
- **Redis** — кэш чёрного списка, троттлинг
- **APScheduler** — периодические задачи (снятие мутов, сбор дохода бизнесов)
- **Docker + Docker Compose** — контейнеризация
- **Oracle Cloud Free Tier** — бесплатный хостинг (ARM, 4 OCPU, 24 GB RAM)

## 📦 Быстрый старт

### 1. Клонирование и настройка
```bash
git clone <repo>
cd moder_bpm_prime
cp .env.example .env
# Отредактируй .env — вставь свои токены
```

### 2. Запуск локально (Docker)
```bash
docker compose up -d --build
docker compose exec bot alembic upgrade head
docker compose exec bot python scripts/init_db.py
```

## 📋 Команды

### Пользовательские
| Команда | Описание |
|---------|----------|
| `/start` | Регистрация / главное меню |
| `/profile` | Профиль (баланс, уровень, работа, клан) |
| `/balance` | Баланс |
| `/top [money\|level]` | Топ игроков |
| `/daily` | Ежедневная награда |
| `/streak` | Текущий стрик |
| `/work` | Работа (выбор, выполнение) |
| `/crime` | Преступления |
| `/business` | Бизнесы (покупка, сбор, апгрейд) |
| `/clan` | Кланы (создание, вступление, инфо, топ) |
| `/shop` | Магазин |
| `/buy <id> [кол-во]` | Купить предмет |
| `/inventory` | Инвентарь |
| `/help` | Помощь |

### Модерация (админы)
| Команда | Описание |
|---------|----------|
| `/ban @user [время] [причина]` | Бан |
| `/unban @user` | Разбан |
| `/mute @user <время> [причина]` | Мут (10m, 1h, 1d) |
| `/unmute @user` | Размут |
| `/warn @user [причина]` | Варн (3 = авто-бан) |
| `/unwarn @user` | Снять варн |
| `/blacklist add <слово> [действие] [время]` | Добавить в ЧС |
| `/blacklist del <слово>` | Удалить из ЧС |
| `/blacklist list` | Список ЧС |
| `/blacklist toggle <слово>` | Вкл/выкл |

### Супер-админ
| Команда | Описание |
|---------|----------|
| `/stats` | Статистика бота |
| `/broadcast <текст>` | Рассылка всем |
| `/give @user <money\|exp> <кол-во>` | Выдать ресурсы |
| `/setlevel @user <уровень>` | Установить уровень |
| `/reload_blacklist` | Обновить кэш ЧС |
| `/userinfo @user` | Инфо о пользователе |
| `/logs @user [лимит]` | Логи модерации |

## 📁 Структура проекта
```
moder_bpm_prime/
├── bot/                    # Основной код бота
│   ├── handlers/           # Хендлеры команд
│   ├── middlewares/        # Middleware (ЧС, троттлинг, регистрация)
│   ├── services/           # Бизнес-логика (фарм, модерация, кланы)
│   ├── database/           # Модели, CRUD, подключение
│   ├── keyboards/          # Клавиатуры
│   ├── utils/              # Утилиты (нормализация, форматирование)
│   └── main.py             # Точка входа
├── scripts/                # Скрипты (деплой, бэкап, сиды)

├── docs/                   # Документация
```
