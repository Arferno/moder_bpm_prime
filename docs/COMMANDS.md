# Список всех команд

## 👤 Пользовательские команды

### Профиль и статистика
| Команда | Описание | Пример |
|---------|----------|--------|
| `/start` | Регистрация / главное меню | `/start` |
| `/profile` | Полный профиль | `/profile` |
| `/balance` | Только баланс | `/balance` |
| `/top [money\|level]` | Топ игроков | `/top money` |
| `/mystats` | Подробная статистика | `/mystats` |

### Фарминг
| Команда | Описание | Пример |
|---------|----------|--------|
| `/daily` | Ежедневная награда (+стрик) | `/daily` |
| `/streak` | Текущий стрик | `/streak` |
| `/work` | Меню работ (inline кнопки) | `/work` |
| `/jobinfo <id>` | Инфо о работе | `/jobinfo 3` |
| `/crime` | Меню преступлений | `/crime` |
| `/crimeinfo <id>` | Инфо о преступлении | `/crimeinfo 2` |
| `/business` | Меню бизнесов | `/business` |
| `/businessinfo <id>` | Инфо о бизнесе | `/businessinfo 

### Кланы
| Команда | Описание | Пример |
|---------|----------|--------|
| `/clan` | Инфо о своём клане / топ кланов | `/clan` |
| `/clan create <name> <tag>` | Создать клан (100к$) | `/clan create "Elite" ELT` |
| `/clan join <tag>` | Вступить в клан | `/clan join ELT` |
| `/clan leave` | Покинуть клан | `/clan leave` |
| `/clan info [tag]` | Инфо о клане | `/clan info ELT` |
| `/clan top` | Топ кланов | `/clan top` |
| `/clan members` | Участники клана | `/clan members` |
| `/clan treasury` | Казна клана | `/clan treasury` |

### Магазин и инвентарь
| Команда | Описание | Пример |
|---------|----------|--------|
| `/shop [категория]` | Магазин (all/boost/protection/...) | `/shop boost` |

## 🛡 Команды модерации (только админы)

### Базовые
| Команда | Описание | Пример |
|---------|----------|--------|
| `/ban @user [время] [причина]` | Бан пользователя | `/ban @user 1d Спам` |
| `/unban @user` | Разбан | `/unban @user` |
| `/mute @user <время> [причина]` | Мут | `/mute @user 10m Мат` |
| `/unmute @user` | Размут | `/unmute @user` |
| `/warn @user [причина]` | Варн (3 = бан) | `/warn @user Оскорбление` |
| `/unwarn @user` | Снять варн | `/unwarn @user` |

### Форматы времени
- `30s` — 30 секунд
- `10m` — 10 минут
- `1h` — 1 час
- `1d` — 1 день
- `7d` — 7 дней

### Чёрный список
| Команда | Описание | Пример |
|---------|----------|--------|
| `/blacklist` | Меню ЧС | `/blacklist` |
| `/blacklist add <слово> [действие] [время]` | Добавить слово | `/blacklist add реклама mute 1h` |
| `/blacklist del <слово>` | Удалить слово | `/blacklist del реклама` |
| `/blacklist list` | Список слов | `/blacklist list` |
| `/blacklist toggle <слово>` | Вкл/выкл слово | `/blacklist toggle мат` |

**Действия:** `delete` (только удалить), `warn` (по умолчанию), `mute`, `ban`

## ⚙️ Супер-админ команды (только SUPER_ADMIN_ID)

| Команда | Описание | Пример |
|---------|----------|--------|
| `/stats` | Статистика бота | `/stats` |
| `/broadcast <текст>` | Рассылка всем | `/broadcast Обновление!` |
| `/give @user <money\|exp> <кол-во>` | Выдать ресурсы | `/give @user money 10000` |
| `/setlevel @user <уровень>` | Установить уровень | `/setlevel @user 50` |
| `/reload_blacklist` | Обновить кэш ЧС | `/reload_blacklist` |
| `/userinfo @user` | Инфо о пользователе | `/userinfo @user` |
| `/logs @user [лимит]` | Логи модерации | `/logs @user 50` |

## 🎮 Reply-клавиатура (в личке с ботом)
При нажатии `/start` в личке появляется клавиатура:
- 👤 Профиль
- 💰 Баланс
- 📊 Топ
- 🎁 Ежедневка
- 💼 Работа
- 🔫 Преступления
- 🏢 Бизнес
- 🏰 Клан
- 🛒 Магазин
- 🎒 Инвентарь
- ❓ Помощь

## 📝 Примеры использования

### Добавить слово в ЧС с мутом на 1 час
```
/blacklist add казино mute 1h
```

### Забанить на неделю
```
/ban @spammer 7d Реклама казино
```

### Выдать 1 млн $ (супер-админ)
```
/give @user money 1000000
```

### Рассылка с HTML
```
/broadcast <b>Внимание!</b> Завтра технические работы с 03:00 до 05:00.
```

### Создать клан
```
/clan create "BPM Elite" BPM
```

### Купить 5 энергетиков
```
/buy 1 5
```
