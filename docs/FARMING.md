### Механика
1. **Успех** (по `success_rate`):
   - Деньги: `random(min_money, max_money)`
   - XP: `exp_reward`
   - **5% шанс** быть пойманным даже при успехе → тюрьма

2. **Неудача**:
   - Тюрьма: `random(jail_time_min, jail_time_max)`
   - Деньги: 0, XP: 0

3. **В тюрьме**:
   - Нельзя работать (`/work`)
   - Нельзя совершать преступления (`/crime`)
   - Можно: `/daily`, `/business collect`, `/shop`, `/clan`
   - Выход: ждать, использовать "Ключ от тюрьмы" (25к$25к$), ждать истечения

Лвл 2: ×2 = 3,000 $/ч

## 🏰 Кланы

### Создание
- Цена: **100,000 $**
- Тег: 2-16 символов (уникальный)
- Владелец = создатель

### Уровни клана
```
XP для клан-уровня N = N² × 5,000
```

| Лвл клана | XP | Перки |
|-----------|-----|-------|
| 1 | 0 | Базовый |
| 2 | 20,000 | +5% к зарплатам участников |
| 3 | 45,000 | +10% к ежедневке |
| 4 | 80,000 | +10% к доходу бизнесов |
| 5 | 125,000 | Свой клановый предмет в магазине |
| 10 | 500,000 | Элитный клан |

### Доход казна
```
Базовый доход/час = лвл_клана × 1,000 $
+ 50 $/час за каждого участника
```

Распределяется раз в час автоматически (APScheduler).

### Роли
| Роль | Права |
|------|-------|
| Участник | Базовые |
| Офицер | Приглашать/кикать, управлять казнами |
| Владелец | Всё + передача владения, распуск клана |

## ⚖️ Баланс экономики

### Источники денег (инфляция)
1. `/work` — основной
2. `/crime` — рискованный
3. `/business` — пассивный (на удалёнке)
4. `/daily` — гарантированный
5. Магазин (лотерея, предметы) — случайный

### Стоки денег (дефляция)
1. Покупка бизнесов
2. Апгрейд бизнесов
3. Создание кланов (100к)
4. Магазин (предметы)
5. Штрафы за ЧС (если добавить)

### Контроль инфляции
- ROI бизнесов фиксирован (41.7 дней)
- Кулдауны работ растут с уровнем
- Преступления имеют высокий риск тюрьмы
- Ежедневка ограничена стриком

### Рекомендации по балансу
- Не снижай цены бизнесов ниже 1000h ROI
- Не повышай success_rate преступлений выше 50%
- Следи за суммой денег в экономике (`/stats`)
- При необходимости — вводи налоги или ивенты сжигания денег
